# Photos

Drop member and PI headshots here (or upload them through the Pages CMS, which
saves them to this folder automatically).

## How photos appear on the site

- **PI headshot** — set the "PI — photo" field on the People page in the CMS, or
  add `"pi_photo": "images/people/bini.jpg"` to `content/people.json`.
- **Team members** — each person has a "Photo" field; set it to the image path,
  e.g. `images/people/matt-allen.jpg`. When empty, the tile shows the persons
  initials, so partial coverage looks intentional.

## Guidance

- Square images look best (tiles are 1:1). ~600x600 px is plenty.
- Use `.jpg` or `.webp` for photos, `.png` for logos.
- Keep filenames lowercase with hyphens: `richard-liu.jpg`.
